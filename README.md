# JNExRate

React App to show exchange rate (along to a TradingView chart) and calculate conversion of some of the most important currencies in the world.

Available at [jnexrate.veest.net](https://jnexrate.veest.net)

## Installation

```bash
$ npm install
```

## Available Scripts

```bash
$ npm run dev
```

```bash
$ npm run build
```

```bash
$ npm run preview
```
